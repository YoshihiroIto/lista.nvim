Install
-------------------------------------------------------------------------------

```vim
Plug 'lambdalisue/vim-lista'
```

Usage
-------------------------------------------------------------------------------
Execute `:Lista` and use `<C-n>/<C-p>` to select a candidate and hit `<CR>` to jump.
